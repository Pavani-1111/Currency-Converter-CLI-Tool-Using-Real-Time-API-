#Currency Converter CLI Tool

This is a simple Python command-line tool that converts an amount from one currency to another using real-time exchange rates from the ExchangeRate-API.

## How It Works
1. The user inputs the amount, source currency, and target currency.
2. The script fetches the latest exchange rate using a free API.
3. It calculates and displays the converted amount.

##  Technologies Used
- Python 3
- `requests` library
- ExchangeRate-API
